Vue-cli build in Windows
更新于 2020-03-05 Vue Node 热度 1 喜欢 0

环境搭建

安装Node.js

从Node.js官网上下载 Node.js for Windows(x64)的文件，一路“下一步”安装完成。

注：

  • 使用Win+R输入cmd打开命令行输入node -v，出现版本号，则说明安装成功。

  • 包管理工具npm在安装Node.js时会一起安装好。

安装Webpack

在命令行中输入npm install webpack -g。

注：

  • 安装完成后webpack -v，出现版本号，则说明安装成功。

  • 若出现如下报错，执行npm install webpack-cli -D，安装webpack-cli。

安装Vue-cli

在命令行中输入npm install vue-cli -g。

注：

  • 安装完成后输入vue -V(大写V)，出现版本号，则说明安装成功。
