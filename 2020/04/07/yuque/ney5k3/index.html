Windows下安装Vue-cli
更新于 2022-09-11

环境搭建

安装 Node.js

从Node.js 官网上下载 Node.js for Windows(x64)的文件，一路“下一步”安装完成。
注：

  • 使用 Win+R 输入 cmd 打开命令行输入node -v，出现版本号，则说明安装成功。

  • 包管理工具npm在安装 Node.js 时会一起安装好。

安装 Webpack

在命令行中输入npm install webpack -g。

注：

  • 安装完成后webpack -v，出现版本号，则说明安装成功。
  • 若出现如下报错，执行npm install webpack-cli -D，安装 webpack-cli。

安装 Vue-cli

在命令行中输入npm install -g @vue/cli。

注：

  • 安装完成后输入vue -V(大写 V)，出现版本号，则说明安装成功。
